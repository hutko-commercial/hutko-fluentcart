hutko Payment Gateway for FluentCart.

== Description ==

Accept credit cards, Apple Pay, Google Pay and other payment methods via hutko on your FluentCart store. 
Supports hosted and embedded checkout, refunds and subscriptions.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` directory, or install through WordPress admin.
2. Activate the plugin.
3. Go to FluentCart Settings → Payments → Hutko and configure your Merchant ID and Secret Key.
